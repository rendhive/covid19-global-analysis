# COVID-19 Global Analysis

## Deskripsi
Project analisis data COVID-19 global untuk melihat:
- Tren kasus
- Tren kematian
- Perbandingan vaksinasi antar negara

## Cara Menjalankan
1. Clone repository
2. Install dependencies:
	

pip install -r requirements.txt

3. Jalankan Jupyter Notebook:
	
jupyter notebook notebooks/covid_analysis.ipynb

## Insight yang Didapat
- Tren kasus global dari waktu ke waktu
- Negara dengan kasus tertinggi
- Perbandingan vaksinasi antar negara
